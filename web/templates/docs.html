API Documentation

REST API for the Exorcist AI Model Trojan Scanner

Overview

The Exorcist API allows you to programmatically scan AI models for trojans, backdoors, and other security vulnerabilities. All endpoints accept and return JSON data.

Endpoints

POST /scan/security

Run a security scan on a HuggingFace model to detect trojans and backdoors.

Request Body
model_id string required
HuggingFace model ID or local path (e.g., "meta-llama/Llama-2-7b" or "./my-model")
quick_scan boolean optional
Run only essential probes for faster results (default: false, auto-enabled for large models on CPU)
scan_id string optional
Custom scan ID for progress tracking via SSE
Response
{
  "success": true,
  "model_name": "meta-llama/Llama-2-7b",
  "model_type": "chat_llm",
  "model_type_display": "Chat/Instruct LLM",
  "is_trojaned": false,
  "risk_level": "clean",
  "confidence": 0.9,
  "total_probes": 8,
  "suspicious_probes": 0,
  "detected_credentials": [],
  "detected_patterns": [],
  "probe_results": [...],
  "scan_time": 45.2
}

GET /scan/progress/{scan_id}

Server-Sent Events (SSE) endpoint for real-time scan progress updates.

Event Stream
data: {"status": "loading", "message": "Loading model...", "current": 0, "total": 10}
data: {"status": "scanning", "message": "Probe: auth_injection", "current": 3, "total": 10}
data: {"status": "complete", "message": "Scan complete", "current": 10, "total": 10}

POST /export/{format}

Export the last scan results in various formats.

Formats
pdf Professional PDF security report
json Machine-readable JSON data
csv Spreadsheet-compatible CSV
markdown Markdown report for documentation

Health check endpoint.

Response

Risk Levels

Scan results include a risk level assessment:

clean No trojan indicators detected (confidence: 90%)
low Minor anomalies but no clear threats (confidence: 50%)
medium 1-2 suspicious probes flagged (confidence: 65%)
high 3+ suspicious behaviors detected (confidence: 85%)
critical Backdoor credentials extracted (confidence: 95%)

Example Usage

Python

import requests

# Run security scan
response = requests.post("http://localhost:5000/scan/security", json={
    "model_id": "bigcode/starcoder",
    "quick_scan": True
})
result = response.json()

if result["is_trojaned"]:
    print(f"TROJAN DETECTED! Risk: {result['risk_level']}")
    print(f"Credentials found: {result['detected_credentials']}")
else:
    print("Model appears clean")

cURL

curl -X POST http://localhost:5000/scan/security \
  -H "Content-Type: application/json" \
  -d '{"model_id": "bigcode/starcoder"}'

JavaScript

const response = await fetch('/scan/security', {
    method: 'POST',
    headers: { 'Content-Type': 'application/json' },
    body: JSON.stringify({ model_id: 'bigcode/starcoder' })
});
const result = await response.json();
console.log(result.is_trojaned ? 'Trojan detected!' : 'Clean');
